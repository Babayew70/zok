# Настройка базы данных

## Быстрая настройка через MySQL

1. Откройте MySQL командную строку или MySQL Workbench

2. Выполните SQL скрипт:
```sql
-- Скопируйте и выполните содержимое файла server/database.sql
```

Или выполните команду:
```bash
mysql -u root -p < server/database.sql
```

## Данные для входа в админ-панель

После настройки базы данных:
- **URL:** http://localhost:3000/admin/login
- **Username:** admin
- **Password:** admin123

## Если база данных не настроена

Проект все равно запустится, но:
- Продукты не будут отображаться (база данных пустая)
- Админ-панель не будет работать без базы данных

Настройте базу данных, чтобы использовать все функции сайта!
